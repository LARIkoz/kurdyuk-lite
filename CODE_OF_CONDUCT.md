# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1.

In short: be respectful, assume good faith, and keep discussion focused on the work. Harassment, personal attacks, and other unwelcome conduct are not tolerated, in issues, pull requests, or any other project space.

## Reporting

If you experience or witness unacceptable behavior, report it privately to the maintainers via GitHub — open a [private security advisory](https://github.com/LARIkoz/kurdyuk-lite/security/advisories/new) (it is confidential and reaches the maintainers directly) or contact a maintainer through their GitHub profile. Reports are reviewed and handled in confidence.

The full text of the standards and enforcement guidelines is in the linked Contributor Covenant.
